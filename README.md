# MacDivvun.service

## Requirements

- Xcode
- autoconf
- automake
- CocoaPods
- cmake
- gettext
- libtool
- pkg-config
- python3

## Building

Run the following commands:

```bash
pod install
git submodule update --init
```

Then, open `MacDivvun.xcworkspace` and build. Easy.

## Installing

Pop into `~/Library/Services`.
